# 📂 Multer Practice

A simple Node.js and Express project demonstrating how to upload files using **Multer**. This project covers the basics of handling file uploads, storing them on the server, and serving uploaded files.

---

## 🚀 Features

- Upload single files
- Upload multiple files
- Store files locally
- Static file serving
- Simple Express server setup
- Beginner-friendly project structure

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- Multer
- HTML/CSS
- JavaScript

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Jitendra-Jakkula/multer-practice.git
```

Go to the project directory

```bash
cd multer-practice
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm start
```

or

```bash
node app.js
```

---

## 📤 Usage

1. Open your browser.
2. Visit

```
http://localhost:3000
```

3. Select a file.
4. Upload it.
5. The uploaded file will be stored inside the configured uploads folder.
